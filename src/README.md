# Project Name Source Code

## Overview
This file contains the runtime logic and resources required to run the 
project FlockingSimulator.

## Requirements
- Unity 2022.3.62f3

## How to Run
1. Clone repo
2. Open in Unity Hub
3. Open Scenes/SampleScene.unity
4. Press Play

## Project Structure
```text
Project/
├── Assets/
│   ├── Images/                 # Images used by UI
│   ├── Material/               # Materials used by Objects
│   ├── Prefabs/                # Prefabs defined as Obstacles
│   ├── Scenes/                 # The project main scenes
│   ├── Scripts/                # The project main logic
│   ├── Settings/               # The project data configurations
│   ├── TextMesh Pro/           # The font resources used by UI
│   └── TestDatas/              # The initial data used in functional requirement tests
│   └── Tests/                  # The project test logic and functional test data
├── Packages/                   # Packages the project use
├── ProjectSettings/            # Engine-wide configuration settings
└── README.md
```
